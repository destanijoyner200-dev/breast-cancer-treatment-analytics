# Breast Cancer Treatment Analytics Dashboard

This project analyzes breast cancer patient data to identify treatment patterns and clinical trends using Power BI.

## Tools Used
Power BI  
DAX  
Excel  
Healthcare Data Visualization

## Project Overview
The dataset contains clinical information on breast cancer patients including age at diagnosis, tumor stage, receptor subtype, surgery type, and chemotherapy treatment.

Using Power BI, the data was cleaned, modeled, and transformed into an interactive dashboard to support healthcare analytics and clinical insight generation.

## Key Features

• KPI metrics including total patients, mastectomy share, HER2 negative mastectomy share, and chemotherapy rate  
• Age group analysis using calculated columns  
• Treatment comparisons by tumor stage and surgery type  
• Receptor subtype distribution visualization (ER / PR / HER2)  
• Interactive filters for exploring patient demographic and clinical patterns  

## Dataset
Breast cancer clinical dataset including demographic and treatment variables used for dashboard analysis.
